# Hello World

Your task will be to create hello world program and then fancy it up a little bit.

1. Initialize node project with npm
2. write Hello world! in the console
3. Create function getGreeting that returns greeting string. In our case: Hello world!, then invoke this function in the main flow.
4. Add parameter to the function called name, and return string with following format `Hello <name>!`
5. Add condition to the function that if name is not provided then return `Hello world!`
